# 🦅 pka2xml: Ultra-Vendored Reversing Kit

> **"Eagles have no borders! Eagle fly free!"** 🦅🦅🦅
>
> 이 프로젝트는 Cisco Packet Tracer의 `.pka`, `.pkt` 파일을 리버싱하여 XML로 변환해주는 강력한 도구입니다. 
> 모든 의존성이 프로젝트 내부에 **'울트라 벤더링'**되어 있어, 인터넷 연결 없이도 완벽한 빌드와 실행을 보장합니다.

---

## 🌟 프로젝트 특징
- **100% 자급자족**: 모든 C++, Python, GUI 의존성이 포함되어 있습니다. (`node_modules`까지!)
- **초보자 친화적**: 복잡한 빌드 과정 없이 전용 스크립트로 누구나 쉽게 사용할 수 있습니다.
- **프리미엄 GUI**: 세련된 다크 모드 인터페이스와 실시간 로그, 시각화 기능을 제공합니다.

---

## 🚀 아주 쉬운 사용 방법

### 1단계: 초기 설정 (처음 한 번만!)
GitHub의 용량 제한으로 인해 분리되어 있는 대용량 파일들을 재조립합니다.
```bash
./setup.sh
```

### 2단계: 골라서 실행하기

#### **옵션 A: 세련된 GUI로 사용하고 싶다면?**
가장 추천하는 방법입니다. 화려한 화면에서 버튼 하나로 모든 걸 해결하세요!
```bash
./run_gui.sh
```

#### **옵션 B: 강력한 CLI(터미널)로 사용하고 싶다면?**
기존 리버서들의 스타일대로 터미널에서 직접 명령어를 입력하세요.
```bash
./run_cli.sh -d [원본파일.pka] [결과파일.xml]
```

---

## 🖥️ GUI 주요 기능 안내
- **파일 선택**: 드래그 앤 드롭으로 `.pka` 파일을 간편하게 불러오세요.
- **실시간 로그**: 복호화 과정이 실시간 터미널 뷰로 투명하게 공개됩니다.
- **원클릭 액션**: 복호화부터 암호화, 버전 패치까지 버튼 클릭 한 번이면 끝납니다.
- **시각화 엔진**: 네트워크 구성도를 시각적으로 확인할 수 있어 분석이 수월합니다.

---

## 🛠️ CLI 고급 사용법
```bash
./run_cli.sh -d [원본.pka] [결과.xml]    # 복호화
./run_cli.sh -e [원본.xml] [결과.pka]    # 암호화
./run_cli.sh -f [원본.pkt] [결과.pkt]    # 버전 고정 패치
./run_cli.sh --forge [인증파일]         # 인증 토큰 생성
```

## 🛡️ 울트라 벤더링 테크니컬 리포트
이 프로젝트는 **GitHub LFS 없이** 모든 대용량 의존성을 담는 혁신을 이뤄냈습니다.
1. **바이너리 분할**: 100MB를 넘는 Electron 및 CryptoPP 라이브러리를 50MB 단위로 쪼개어 Git에 저장했습니다.
2. **Yarn Zero-Installs**: GUI 의존성 전체를 ZIP 캐시와 `node_modules` 형태로 포함하여 `npm install` 과정이 필요 없습니다.
3. **오프라인 Python**: 필요한 모든 라이브러리를 Wheel 형태로 보관합니다.
